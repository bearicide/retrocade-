# RETROCADE Portable Pixel Asset Pack

Original monochrome handheld-style assets for RETROCADE.

## Palette

- Ink: `#081820`
- Dark: `#346856`
- Mid: `#88c070`
- Light: `#e0f8d0`
- Accent: `#c8e890`

## Files

- `retrocade-portable-pixel-sprite-pack.svg` - source sprite sheet with cabinet sprites, tiles, palette swatches, and UI pieces.
- `../../css/portable-pixel-pack.css` - visual layer used by the shared game engine.
- `../../js/working-games.js` - shared game engine that renders the sprites and palette.

## Cabinets covered

Snake, Breakout, Invaders, Space Rocks, Pong, Claw Machine, Bubble Shooter, Memory Match, Mini Racer, Rhythm Pads, Falling Blocks, and Frog Dash.

## Notes

These are original RETROCADE assets inspired by portable monochrome pixel hardware. They are not ripped, traced, or copied from commercial games.
